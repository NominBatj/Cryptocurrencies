# Cryptocurrencies
## Overview 

In this project we are creating analysis for clients who are preparing to enter the cryptocurrency market. The analysis includes cryptocurrencies in the trading market and how they can be grouped to create a classification system. To reduce overfitting, Principal Component Analysis (PCA) was used.
